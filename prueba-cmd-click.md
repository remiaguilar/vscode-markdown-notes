# Prueba de Wiki-Links con Cmd+Click

Ahora puedes usar **cmd+click** en macOS para navegar a los wiki-links:

- [[estaesunanuevanota]] ← Haz cmd+click aquí
- [[otra nota de prueba]] ← O aquí
- [[archivo con espacios]] ← También funciona con espacios
- [[config importante.json]] ← Y con extensiones específicas

## Instrucciones:

### En macOS:
- **Cmd+Click**: Para navegar al archivo (se crea automáticamente si no existe)
- **F12**: También funciona para ir a la definición

### En Windows/Linux:
- **Ctrl+Click**: Para navegar al archivo
- **F12**: Para ir a la definición

### ✨ Funcionalidades:
- **Creación automática**: No pregunta, crea el archivo directamente
- **Soporte para espacios**: Los nombres con espacios funcionan perfectamente
- **Extensiones específicas**: Puedes especificar .js, .json, .py, etc.
- **Navegación fluida**: Se crea y abre inmediatamente
