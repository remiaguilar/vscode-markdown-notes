# Ejemplo de Wiki Links

Esta es una demostración de cómo funcionan los wiki-links en la extensión:

## Ejemplos de uso:

1. **Link a otra nota markdown**: [[mi-otra-nota]]
2. **Link a archivo con extensión específica**: [[config.json]]
3. **Link a archivo JavaScript**: [[script.js]]
4. **Link a archivo Python**: [[main.py]]
5. **Link a archivo HTML**: [[index.html]]
6. **Link a archivo CSS**: [[estilos.css]]
7. **Link a archivo de texto**: [[notas.txt]]

## Cómo usar:

- **Hover**: Mantén el cursor sobre cualquier `[[link]]` para ver información
- **F12**: Presiona F12 cuando el cursor esté sobre un link para navegar
- **Ctrl+Click**: Ctrl+Clic para navegar al archivo
- **Creación automática**: Si el archivo no existe, se te preguntará si quieres crearlo

## Búsqueda inteligente:

Los wiki-links buscan archivos en todo el árbol de directorios, por lo que no necesitas especificar la ruta completa.
